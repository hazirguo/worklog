worklog
=======

> The log of everyday's work, study, entertainment and etc.

### 面临的窘境

* 时间管理能力比较差
* 学习效率较低
* 容易分心


### 解决方法

* 每天要有个 ToDo list 的计划
* 每天要简单总结是否完成了预期目标

### 组织形式

* 将当前周的每天 ToDo list 以每个 `XX-XX.md` 文件名放在根目录下
* 每周结束之后，做个小结，并将该周的所有文件打包到一个文件夹下，以 `XX.XX~XX.XX` 命名文件夹
* 每月结束之后，做个小结，并将该月的所有文件夹打包到一个文件夹下，以 `monthname` 命名文件夹
